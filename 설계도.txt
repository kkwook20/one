2. 주요 기능 구현 계획
2.1 노드 공통 기능

상단 버튼: 실행/중지, 삭제, 드롭다운 메뉴
작업 항목 리스트: 드래그&드롭, 상태 아이콘 (○/×/△)
진행률 표시: 실시간 업데이트
버전 관리: 최대 5개 히스토리 자동 저장

2.2 Worker Node 계열

Worker Node: 3패널 구조, AI 자동 개선
Worker Writer Node: 텍스트 생성, 마크다운/JSON 출력
Worker Painter Node: 이미지 생성, SVG/픽셀 지원

2.3 매니저 노드 강화

Supervisor: 코드 리뷰, 수정 이력, 자동 백업
Planner: 평가 패널, Accept/Cancel 버튼
Watcher: 요청 처리, 데이터 수집, 출처 추적
Scheduler: 간트차트, 시간 추적, AI 예측
Flow: 실행 팝업, 매니저 우선 실행
Storage: 폴더뷰, 데이터 흐름 그래프

2.4 글로벌 변수 시스템

네이밍: {섹션}.{노드타입}.{노드ID}.{데이터타입}.{항목}
자동 완성 지원
타입 체크 및 검증
실시간 참조 업데이트

2.5 파일 저장 구조
node-storage/
├── {node-id}/
│   ├── code/
│   │   ├── current.py
│   │   └── history/
│   │       ├── v1.py
│   │       ├── v2.py
│   │       └── ... (최대 5개)
│   ├── data/
│   │   ├── input.json
│   │   ├── output.json
│   │   └── tasks.json
│   ├── files/
│   │   └── [생성된 파일들]
│   └── metadata.json
