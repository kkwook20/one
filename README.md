# Workflow Engine

A powerful, node-based workflow automation engine with AI integration, real-time monitoring, and visual programming capabilities.

![Workflow Engine Banner](docs/images/banner.png)

## 🚀 Features

- **Visual Workflow Editor**: Intuitive drag-and-drop interface with React Flow
- **Multiple Node Types**: Worker, Supervisor, Planner, Watcher, Scheduler, Flow, and Storage nodes
- **AI Integration**: Support for OpenAI, Anthropic, and local LLMs
- **Real-time Monitoring**: System metrics, performance analytics, and error tracking
- **External Tool Integration**: Maya, Houdini, and Unreal Engine connectors
- **Template System**: Pre-built workflows for common tasks
- **WebSocket Communication**: Real-time updates and execution status
